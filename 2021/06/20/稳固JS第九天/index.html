稳固JS第九天（房子问题，三天合一天）

  1. PC端网页特效

    • 元素偏移量offset系列

      • 获取元素距离带有定位父元素的位置

      • 获取元素自身的大小（宽度高度）

      • 注意：返回的数值都不带单位

      • offset系列常用属性：

        • element.offsetParent：返回作为该元素带有定位的父级元素
        • elment.offsetTop：返回元素带有定位父元素上方的偏移量
        • element.offsetLeft：返回元素带有定位父元素左边框的偏移量
        • element.offsetWifth：返回自身，包括padding、边框、内容区的宽度，返回值不带单位
        • element.offsetHeight：返回自身，包括padding、边框、内容区的高度，返回值不带单位

      • offset和style的区别

        1. offset可以得到任意样式表中的样式值，而style只能得到行内样式的样式值
        2. offset系列获得的数值是没有单位的，而style的到的是一个字符串类型的值
        3. offsetWidth包含padding+border+width，而style只获得width的值
        4. offsetWidth等属性是只读属性，只能获取不能赋值，style.width是可读写的
        5. 所以我们想要获取元素大小位置，用offset合适，想给元素修改值，用style更合适

    • 元素可视区client系列

      • element.clientTop：返回元素上边框的大小
      • element.clientLeft：返回元素左边框的大小
      • element.clientWidth：返回自身包括padding、内容区的宽度、不含边框，返回值不带单位
      • element.clientHeight：返回自身包括padding、内容区的高度、不含边框，返回值不带单位

    • 元素滚动scroll系列

      • element.scrollTop：返回被卷去的上侧距离（滚动后，页面上部被隐藏的距离），返回值不带单位

      • element.scrollLeft：返回被卷去的左侧距离，返回值不带单位

      • element.scrollWidth：返回自身实际的高度，不含边框，返回数值不带单位

      • element.scrollHeight：返回自身实际的高度，不含边框，返回值不带单位

      • onscroll事件：拖动滚动条时触发

      • 三大系列总结：

        1. offset系列 经常用于获得元素位置
        2. client系列 经常用于获取元素大小
        3. scroll系列 经常用于获取滚动距离

    • 动画函数封装

      • 动画实现原理

        • 核心原理：通过定时器setInterval（）不断移动盒子位置

      • 简单动画函数封装

        • 需要传递的两个参数，动画对象和移动到的距离

      • 动画函数给不同元素记录不同的定时器

      • 缓动动画原理

        • 就是让元素运动速度有所变化，最常见的是让速度慢慢停下来
        • 思路：
          • 让每次移动距离慢慢变短
          • 核心算法：（目标值-现在的位置）/10 作为每次移动的距离 步长
          • 停止条件是：让当前盒子位置等于目标位置就停止定时器
          • 步长值要取整（Math.ceil ( ) ）
          • Tips：正值可以按上面这么取，但是负值，需要往小的取整（Math.floor ( ) ）

      • 动画函数添加回调函数

        • 回调函数写的位置：定时器结束的位置

      • 将动画函数封装到单独JS文件里面

        • 单独新建一个JS文件

    • 常见网页特效案例

      • 网页轮播图（明天满课实训，明天晚上弄）

    • 补充知识点：

      • 立即执行函数

        • 不需要调用，立马能够自己执行的函数

        • 写法：

          • //写法1：
(function(){}()
 //写法2：
 (function(){}())

        • 有多个立即执行函数，要用分号隔开

        • 立即执行函数最大的作用就是，独立创建了一个作用域，里面所有的变量都是局部变量，不会有命名冲突的情况

      • pageshow：重新加载页面触发的事件

        • 和load事件的对比：火狐浏览器中有个特点：“往返缓存”，这个缓存中不仅保存着页面数据，还保存了DOM和JavaScript的状态；实际上是将整个页面保存在内容中，所以load有时候不能触发。这个时候可以用pageshow，这个事件在页面显示时触发，无论页面是否来自缓存。
        • 如果两者都可以触发的情况下，pageshow会在load事件触发后触发，根据事件对象中的persisted来判断是否是缓存中的页面触发的pageshow事件，注意这个事件要给window添加

      • mouseenter和mouseover的区别

        • mosueenter鼠标事件
          • 当鼠标移动到元素上就会触发这个事件
          • 只有鼠标经过自身盒子会触发，所以这个事件不会冒泡
          • 跟这个事件搭配 鼠标离开事件mouseleave 同样不会冒泡
        • mouseover鼠标事件
          • 鼠标经过自身盒子会触发，经过子盒子还会触发
